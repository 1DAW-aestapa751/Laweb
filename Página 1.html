Guía NO Básica para HTML.

Por Álvaro Estévez Aparicio

Html es una herramienta muy útil en el desarrollo de aplicaciones web, esto debido
a que podremos crear nuestras propias páginas web de forma sencilla, para ello, en
esta guía se enseñarán las etiquetas más básicas para trabajar en HTML.


Parte 1: Las etiquetas head y body.


Comenzando por lo básico, las primeras etiquetas que debemos tener en cuenta las etiquetas <html> (que permite saber a nuestro sistema que tipo de lenguaje vamos a usar) y la etiqueta <head>, que nos va a permitir colocar el nombre que aparecerá en la pestaña de la página.

Dentro de <head> podemos añadir metadatos, los cuales aunque no aparecen directamente en la página pero indican datos como palabras clave, nombre del autor, charset para los carácteres que van a aparecer, el idioma... etc. Para esto usamos la etiqueta <meta> y dentro de esta el tipo de metadato.

Finalemente, el resto de contenido de la web se mete en la etiqueta <body>.



Parte 2: Etiquetas básicas para texto.

Etiquetas básicas de texto en HTML
Etiqueta Ejemplo Descripción
<p> <p>Esto es un párrafo</p> Define un párrafo de texto.
<h1> ... <h6> <h1>Título principal</h1> Encabezados de diferentes niveles (de 1 a 6).
<b> <b>Texto en negrita</b> Resalta el texto en negrita (sin énfasis semántico).
<strong> <strong>Importante</strong> Texto importante o con énfasis fuerte.
<i> <i>Texto en cursiva</i> Texto en cursiva (sin énfasis semántico).
<em> <em>Énfasis</em> Resalta con énfasis (semántico).
<u> <u>Subrayado</u> Subraya el texto.
<br> Texto<br>salto de línea Inserta un salto de línea.
<small> <small>Texto pequeño</small> Texto de menor tamaño.
<mark> <mark>Texto resaltado</mark> Resalta el texto como si estuviera marcado.


Parte 3: Tablas

Para hacer una tabla, usaremo las etiqueta <table>, que crea la tabla, y dentro pondremos las etiquetas <tr>, para cada fila de la tabla, y <td> para cada columna dentro de cada fila.



Parte 4: Formularios

Para crear un formulario, usaremos la etiqueta <form>, dentro de la cual, para separar las secciones de nuestro formulario podemos poner las etiquetas <fieldset>, para separar entre secciones, y <legend>, para añadir un titulo a cada sección del formulario.


Etiquetas del formulario
Etiqueta Ejemplo Descripción
<label> <label for="name">PON TU NOMBRE</label> Etiqueta descriptiva que indica la función de un campo de entrada. Se asocia a un input mediante el atributo for.
<input type="text"> <input type="text" id="name"> Crea un campo de texto para introducir datos cortos.
<input type="checkbox"> <input type="checkbox" name="gender"> Permite seleccionar una o varias opciones (casillas de opciones).
<input type="radio"> <input type="radio" name="laos"> Permite seleccionar solo una opción dentro de un grupo (botones de opción).
<select> <select id="place"> ... </select> Crea un menú desplegable con varias opciones.
<option> <option value="sevilla">Sevilla</option> Define una opción dentro de un menú desplegable <select>.
<input type="email"> <input type="email" id="mail"> Campo para ingresar direcciones de correo electrónico. Valida el formato automáticamente.
<input type="password"> <input type="password" id="password"> Campo para ingresar contraseñas. El texto se muestra oculto.
<textarea> <textarea name="message"></textarea> Área de texto de varias líneas para escribir mensajes o comentarios largos.
<input type="color"> <input type="color" id="fcolor"> Permite al usuario seleccionar un color mediante un selector de color.
<input type="file"> <input type="file" id="file"> Permite subir archivos desde el dispositivo del usuario.
<input type="date"> <input type="date" id="date"> Campo para seleccionar una fecha mediante un calendario.
<input type="search"> <input type="search" id="sch"> Campo de búsqueda optimizado para introducir términos de búsqueda.
<input type="reset"> <input type="reset" value="borrar todo"> Botón que reinicia todos los campos del formulario a sus valores originales.
<input type="submit"> <input type="submit" value="Subir formulario"> Botón que envía los datos del formulario al servidor.


Parte 5: Imágenes e Hiperenlaces

Para decorar nuestras páginas web usando imágenes, debemos usar la etiqueta <img>, y dentro de esta misma pondremos: src="Enlace de la imagen" height="altura en pixeles o porentajes" width="ancho en pixeles o porcentajes".

(Además de enlaces, con src pordemos añadir imagenes de nuestro dispositivo añadiendo la ruta del mismo o poniendo solo el nombre y la extensión del archivo si esta en la misma carpeta que el .html)





Para usar hiperenlaces usamos la etiqueta <a>, y dentro de la etiqueta ponemos: href="enlace", y por último, añadimos un texto o imagen que al clicar nos mande a el enlace.



Parte 6: CSS y Javascript

Para añadir elementos visuales a nuestra página web debemos usar CSS y javascript, aunque no entraré en detalle en esta guía, si queremos añadir algo ocasional podemos usar las etiquetas <style> para añadir algo en CSS, y <script> para añadir algo en javascript.



Parte 7: Canvas

Para crear un canvas en nuestra web, en el que podremos "dibujar dentro" usaremos la etiqueta <canvas>, y dentro añadiremos id="id del canvas" width="ancho en pixeles" height="altura en pixeles". Esto creará en nuestra web el espacio disponible para pintar algo con javascript.

Para ello, usaremos la etiqueta <script> en la que añadiremos el código de javascript para crear cuadrados o rectangulos de colores o poner bordes (Como ya mencioném, en esta página no entraré en explicaciones de CSS o JS).
